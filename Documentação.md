```JavaScript
As funções são blocos de código reutilizáveis que realizam uma tarefa específica. 
Elas são essenciais para a organização e estruturação do código, 
permitindo que você divida o seu programa em partes mais gerenciáveis e reutilizáveis. As funções podem receber parâmetros, executar operações e retornar valores.

Exemplo:

Declaração de Função:

function nomeDaFuncao(parametro1, parametro2) {
  // Código da função
  return resultado; 
}


Parametros da função:
  parametro1, parametro2 // Opcional: retorno da função


Chamada da Função:
  nomeDaFuncao(argumento1, argumento2); 

Retorno da Função:
  return resultado; // Opcional: retorno da função


Os tipos de Funções que existem são:
Funções Nomeadas
Funções com ou sem Retorno
Funções Anonimas
Funções Parametrizadas
Funções Com Parametros REST
Arrow Functions
Funções Geradoras
Funções Aninhadas
Funções de callback
Funções autoinvocadas
```
